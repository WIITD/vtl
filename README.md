# vtl - v terminal launcher
simple terminal based launcher for apps

# config
app can be configured through json files  

example:  
```
{
  "app": "path/to/app"
  "args": [
    "path/to/...",
    "path/to/...",
    ...
  ]
}
```

# building 
to build this app all you need is v compiler

```
v vtl.v
```
